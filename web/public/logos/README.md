# TORIS — Logo Assets

## Arquivos

| Arquivo | Uso |
|---|---|
| `toris-horizontal-dark.svg` | Logo horizontal, texto Charcoal — para fundos claros |
| `toris-horizontal-light.svg` | Logo horizontal, texto branco — para fundos escuros |
| `toris-horizontal-tagline-dark.svg` | Logo horizontal + tagline, texto Charcoal |
| `toris-horizontal-tagline-light.svg` | Logo horizontal + tagline, texto branco |
| `toris-vertical-dark.svg` | Logo vertical + tagline, texto Charcoal |
| `toris-vertical-light.svg` | Logo vertical + tagline, texto branco |
| `toris-icon-sage.svg` | Ícone solo, cor Sage (#7D968C) |
| `toris-icon-white.svg` | Ícone solo, branco |
| `toris-icon-charcoal.svg` | Ícone solo, Charcoal (#1F2A2E) |

## Paleta

- Sage: `#7D968C`
- Charcoal: `#1F2A2E`
- Slate: `#6B7476`
- Soft Grey: `#E9EBEC`
- Off-White: `#F9F9FB`

## Tipografia

Sora — Google Fonts — SemiBold (600) para o logotipo

## Exportar como PNG

Abrir qualquer SVG no Figma, Illustrator ou Inkscape e exportar em 2x ou 3x para alta definição.
